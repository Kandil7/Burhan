# 🪟 Athar Windows Scripts

Quick-launch batch scripts for Windows users.

## 📜 Available Scripts

| Script | What It Does |
|--------|--------------|
| **start.bat** | Start application (API + Frontend) |
| **stop.bat** | Stop all services |
| **test.bat** | Test API endpoints |
| **install.bat** | Install dependencies |
| **menu.bat** | Interactive menu |
| **ingest_data.bat** | Process books/hadith |
| **process_and_embed.bat** | Generate embeddings |

## 🚀 Usage

Just double-click any script to run it!

**Or use the main build system:**
```bash
cd K:\business\projects_v2\Athar
build.bat start
build.bat test
build.bat help
```

## 📋 Note

These are convenience wrappers around `build.bat`. For the full command list, see:
- `build.bat help` (main build system)
- `docs/QUICK_REFERENCE.md` (command cheat sheet)
